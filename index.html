Cuál es la anatomía de una regla CSS?

la anatomia de la regla css se define por un selector q seleciona el elemento HTML, luego esta conjunto a una llaves q tomaran la propiedad y el valor ej: h1 { color: red; font-size: 5em; }

¿Cuáles son los tipos de selectores que puede utilizar CSS?

los tipos de selectores son: Selector de clase Selecciona todos los elementos que tienen el atributo de class especificado. Selector de tipo Selecciona todos los elementos que coinciden con el nombre del elemento especificado Selector de ID Selecciona un elemento basándose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento. Selector universal Selecciona todos los elementos. Opcionalmente, puede estar restringido a un espacio de nombre específico o a todos los espacios de nombres Selector de atributo Selecciona elementos basándose en el valor de un determinado atributo.

¿A qué se llama DOM y cual es su función?

DOM es es esencialmente una interfaz de plataforma que proporciona un conjunto estándar de objetos para representar documentos HTML, XHTML y XML, un modelo estándar sobre cómo pueden combinarse dichos objetos, y una interfaz estándar para acceder a ellos y manipularla
